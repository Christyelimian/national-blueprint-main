## Responsive Guidelines

### Mobile Adaptations

#### **Card Stacking**
- Single column layout
- Metrics stack vertically
- Status badges remain prominent
- Touch-friendly CTA buttons (44px min height)

#### **Color Adjustments**
- Increase contrast for smaller screens
- Maintain status color visibility
- Ensure text readability on all backgrounds

#### **Icon Sizing**
- Scale icons proportionally
- Maintain touch targets (minimum 44px)
- Preserve visual hierarchy
