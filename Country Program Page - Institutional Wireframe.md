## Key Design Principles

### 1. **Institutional Hierarchy**
- Clear information architecture for World Bank/IFC-level audiences
- Executive summary at top, detailed analysis below
- Data-driven metrics prominently displayed

### 2. **Status Differentiation**
- **Active Programs**: Green accent, real-time progress, execution metrics
- **Proposed Programs**: Blue accent, planning status, contingency information

### 3. **Scalable Grid System**
- Programs grid accommodates 1-12+ programs per country
- Filter system for Active vs Proposed viewing
- Consistent card layout across all programs

### 4. **Investor-Friendly Metrics**
- Financial structure transparency
- Risk mitigation clearly articulated
- ESG metrics prominently featured

### 5. **Government Partnership Focus**
- MOU status and terms clearly displayed
- Regulatory approval pathways outlined
- Implementation capacity demonstrated

## Mobile Responsiveness

- Single-column layout on mobile
- Key metrics stack vertically
- Program cards scroll horizontally
- CTA buttons remain thumb-friendly
